データソース・更新頻度

CoinRaderで利用しているデータソース、更新の目安、欠損時の扱いをまとめています。

データソース

  • CoinGecko API（価格、時価総額、出来高、ランキング等） — 公式 / API Docs
各データの権利は提供元に帰属します。

ランキングのルール

  • トレンド：CoinGeckoのトレンド（話題）指標です。価格や出来高の順位ではありません。
  • 上昇率（24h）：ステーブルコインを除外し、24h出来高5億円以上の銘柄を優先して表示します（不足時は出来高条件なしで補完）。
  • 出来高（24h）：24時間出来高（JPY換算）で表示します。
  • 出来高（アルト）：BTC/ETH/ステーブルコインを除外した24時間出来高で表示します。
  • 時価総額：時価総額（JPY換算）で表示します。
※「ステーブルコイン」判定は提供元データを優先し、取得できない場合は一般的なステーブル（例：USDT/USDC/DAI等）を除外対象として扱うことがあります。

更新頻度

  • データは一定間隔で更新されます。ページ上部の「更新」表示をご確認ください。

欠損時の扱い

  • API制限、通信障害、提供元の仕様変更等により一時的にデータが取得できない場合があります。
  • その際は「取得できませんでした」等の表示となる場合があります。
